Mod Merger - desktop copy
=========================

Start: double-click  Start Mod Merger.vbs

Links file:  gamebanana-mods.txt  (in this folder)
Downloads:   mod .zip files save here too (same folder as the app)

Edit gamebanana-mods.txt with direct GameBanana /dl/ links, one per line.
Then use Download mods in the app, or Run Full Merge if you also set a base pack folder in Step 2.
